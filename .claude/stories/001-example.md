# Example Story

## Description

This is an example user story template. Replace this with your actual story.

Create a simple Go function that adds two integers.

## Acceptance Criteria

- [ ] Function `Add(a, b int) int` exists in `pkg/math/add.go`
- [ ] Function returns sum of two integers
- [ ] Function handles negative numbers
- [ ] Function has tests in `pkg/math/add_test.go`

## Definition of Done

- [ ] All tests pass
- [ ] Test coverage > 90%
- [ ] Code compiles without errors
- [ ] Code is formatted with gofmt
