# YNAB

## Description

Дополни систему. Когда транзакции которые нужно загрузить по АПИ уже определены, система пытается соотнести транзакции с аккаунтами из конфига. 

Для транзакций у которых нет аккаунтов, возьми список уникальный кредитных карт (последние 4 цифры):
- для каждой кредитной карты проверь по АПИ если существует аккаунт с таким или похожим названием
- Если есть, возьми его id и сохрани в конфиг
- А если нет, то создай им такой новый аккаунт по апи. После того как аккаунт создан, добавь данные об этом в конфиг.
