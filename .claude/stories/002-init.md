# Initialize the golang app

## Description

Create a golang app skeleton for future development. The skeleton should consider future features:
- reading text messages from a folder
- parsing these messages
- formatting the messages
- sending them to the YNAB API

Basically, it would feed YNAB service with data from text messages.
