# Remove PII

## Description

Этот репозиторий, сторисы и тесты содержат персональные данные из настоящих сообщений которые получал автор. Нужно убрать персональную информацию, сделать ее тестовой. Из репозитория и из истории гит репозитория

## Acceptance Criteria

- [ ] Персональной информации нет в текущем коде
- [ ] В истории и в прошлых коммитов персональной истории тоже нет
- [ ] После всех дел все тесты и покрытие должно соответствовать общим требованиям

## Definition of Done

- [ ] All tests pass
- [ ] Test coverage > 90%
- [ ] Code compiles without errors
- [ ] Code is formatted with gofmt
- [ ] Following go best practices
